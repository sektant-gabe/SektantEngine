This is a template for Imgui C++ projects. What you get\+:


\begin{DoxyItemize}
\item Library and executable code separated in distinct folders.
\item Use of modern CMake for building and compiling.
\item External libraries\+:
\begin{DoxyItemize}
\item All graphical and imgui related libs via Git submodules\+:
\begin{DoxyItemize}
\item GLFW, Imgui and Implot
\end{DoxyItemize}
\item General purpose libraries via CMake Fetch\+Content\+:
\begin{DoxyItemize}
\item \href{https://github.com/nlohmann/json}{\texttt{JSON}}, \href{https://github.com/jarro2783/cxxopts}{\texttt{cxxopts}} and \href{https://github.com/fmtlib/fmt}{\texttt{fmt}}
\end{DoxyItemize}
\end{DoxyItemize}
\item Continuous integration testing with Github Actions and \href{https://pre-commit.com/}{\texttt{pre-\/commit}}
\item Code documentation with \href{https://doxygen.nl/}{\texttt{Doxygen}} and \href{https://franneck94.github.io/CppProjectTemplate/}{\texttt{Github Pages}}
\item Tooling\+: Clang-\/\+Format, Cmake-\/\+Format, Clang-\/tidy, Sanitizers
\end{DoxyItemize}\hypertarget{md_README_autotoc_md1}{}\doxysection{\texorpdfstring{Structure}{Structure}}\label{md_README_autotoc_md1}
